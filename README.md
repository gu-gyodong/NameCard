# NameCard
- 본 프로젝트는 졸업 과제였던 NFC, QR코드를 이용한 명홤 관리 어플을 MVVM디자인 패턴을 이용하여
혼자 다시 구현 해본 프로젝트 입니다

## 구현 기술
- 기본적으로 Android Studio, MySql DB를 이용하여 구현 하였으며 PHP 쿼리문을 이용하여 DB에 연동 하였습니다
